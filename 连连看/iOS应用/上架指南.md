# 《公约数列》上架 App Store 完整指南

## 一、前置条件（你需要先准备）

| 项目 | 说明 | 费用 |
|------|------|------|
| Apple Developer 账号 | 去 developer.apple.com 注册 | **$99/年** |
| Mac 电脑 | 必须用 macOS 才能用 Xcode | — |
| Xcode | 在 Mac App Store 免费下载，约 14GB | 免费 |
| 隐私政策页面 | App Store 强制要求，哪怕 App 不收集数据 | 免费（用 netlify 挂一页就行）|

---

## 二、在 Xcode 里建项目（约10分钟）

1. 打开 Xcode → **Create New Project**
2. 选 **iOS → App** → Next
3. 填写：
   - **Product Name**: 公约数列
   - **Bundle Identifier**: com.你的名字.gongyueshullie（全小写英文，唯一）
   - **Interface**: Storyboard
   - **Language**: Swift
4. 保存到桌面

## 三、把代码拷进去

把 `iOS应用/公约数列/公约数列/` 里的这些文件**替换**掉 Xcode 自动生成的对应文件：

- `AppDelegate.swift` → 替换
- `SceneDelegate.swift` → 替换  
- `GameViewController.swift` → **新增**（右键项目 → Add Files to...）
- `Info.plist` → 替换（或在 Xcode 里手动对照添加键值）
- `Game/index.html` → 拖进 Xcode 项目，勾选 **Copy items if needed**

删掉 Xcode 自动生成的 `ViewController.swift` 和 `Main.storyboard`（不需要了）。

## 四、配置 Xcode

1. 点击项目根节点 → **Signing & Capabilities**
2. 勾选 **Automatically manage signing**
3. Team 选你的 Apple ID
4. Bundle Identifier 填 `com.你的名字.gongyueshullie`

## 五、制作 App 图标

需要一张 **1024×1024 PNG** 图标（无圆角，Apple 自动裁圆）。

推荐用 Canva 或 Figma 做：深紫背景 + "公约数列" 四个大字 + ✨装饰

做好后拖入 `Assets.xcassets → AppIcon` 里对应尺寸格。

## 六、截图（App Store 必须提交）

用 Xcode 模拟器截图，需要这几个尺寸：
- **iPhone 6.7"**（iPhone 15 Pro Max）至少3张
- **iPhone 6.5"**（iPhone 14 Plus）至少3张

Xcode 菜单 → **Product → Run** 运行模拟器，截图保存。

## 七、App Store Connect 配置

1. 打开 [appstoreconnect.apple.com](https://appstoreconnect.apple.com)
2. My Apps → **+** → New App
3. 填写：
   - **Name**: 公约数列 - 数字消消乐
   - **Primary Language**: Chinese Simplified
   - **Bundle ID**: 选你上面创建的
   - **SKU**: gongyueshullie-v1（随便填，唯一就行）
4. 在 **Pricing** 里设置 **$0.99**（最低付费档，苹果叫 Tier 1）
5. 上传截图、填写描述（建议中英双语）

**App 描述示例（英文）：**
```
A fresh twist on classic tile-matching!

Instead of matching identical tiles, connect two numbers 
sharing a common factor — both divide by their GCD. 
If the result is 1, the tile disappears. 
If not, it stays with a new value!

Three difficulty levels with grids from 5×5 to 7×7.
Numbers range up to 200 with primes up to 13.

Features:
• Cute kawaii visual design
• Background music & sound effects  
• Hint system & unlimited undo
• Shuffle when stuck
```

## 八、提交审核

1. 在 Xcode 里 **Product → Archive**
2. Archive 完成后点 **Distribute App → App Store Connect → Upload**
3. 回到 App Store Connect，在 Build 里选刚上传的版本
4. 点 **Submit for Review**
5. 等待 1-3 个工作日（Apple 审核）

## 九、定价与收款

- 定价 $0.99（Tier 1），苹果抽 30%，你到手约 **$0.70/次**
- 收款需要在 App Store Connect → **Agreements, Tax, and Banking** 填银行账户
- 国内用户可填国内银行账号（支持美元汇款）

## 十、审核注意事项

Apple 常见拒绝原因（提前避坑）：
- ❌ 没有隐私政策链接 → 随便用 Netlify 挂一页就行
- ❌ App 崩溃 → 先在真机上测试
- ❌ 截图质量差 → 用模拟器截，不要用照相机拍屏幕
- ❌ 描述与实际功能不符 → 如实填写
